# MacのIPアドレスを確認する方法

MacのローカルネットワークIPアドレスを確認する方法をいくつか紹介します。

## 方法1: システム環境設定から確認（GUI）

1. Appleメニュー（左上のリンゴアイコン）をクリック
2. 「システム環境設定」（または「システム設定」）を選択
3. 「ネットワーク」をクリック
4. 現在接続しているネットワーク（Wi-Fiまたはイーサネット）を選択
5. 右側に表示されるIPアドレスを確認
   - 例: `192.168.1.100`

## 方法2: コマンドラインで確認（ターミナル）

### macOSの標準コマンド

```bash
# すべてのネットワークインターフェースのIPアドレスを表示
ifconfig | grep "inet " | grep -v 127.0.0.1
```

このコマンドの出力例:
```
inet 192.168.1.100 netmask 0xffffff00 broadcast 192.168.1.255
```

### より詳細な情報

```bash
# すべてのネットワークインターフェースの詳細を表示
ifconfig
```

### Wi-FiのIPアドレスだけを確認

```bash
# Wi-Fi（en0）のIPアドレスを確認
ifconfig en0 | grep "inet " | awk '{print $2}'
```

### アクティブなネットワークインターフェースのIPアドレスだけを表示

```bash
# IPv4アドレスだけを取得（内部アドレスを除く）
ipconfig getifaddr en0
```

## 方法3: システム情報から確認

1. Appleメニュー → 「このMacについて」をクリック
2. 「システムレポート」をクリック
3. 左側の「ネットワーク」を展開
4. 「Wi-Fi」または「イーサネット」を選択
5. 右側に表示されるIPアドレスを確認

## 方法4: ターミナルで簡単に確認（推奨）

プロジェクトに含まれているスクリプトを使用：

```bash
node get-local-ip.js
```

このスクリプトは、ローカルネットワークのIPアドレスを取得して、スマホでアクセスするためのURLも表示します。

## 方法5: ネットワーク設定を確認（macOS Monterey以降）

1. Appleメニュー → 「システム設定」をクリック
2. 左側の「ネットワーク」を選択
3. Wi-Fiまたはイーサネットを選択
4. 右側に表示されるIPアドレスを確認

## 📝 よくあるIPアドレスの形式

ローカルネットワークのIPアドレスは通常、以下の範囲にあります：

- `192.168.x.x` （最も一般的）
- `10.0.x.x`
- `172.16.x.x 〜 172.31.x.x`

`127.0.0.1`や`localhost`は自分のマシンを指すローカルループバックアドレスで、他のデバイスからはアクセスできません。

## 🔍 確認コマンドの例

```bash
# すべてのIPv4アドレスを表示（ループバックを除く）
ifconfig | grep "inet " | grep -v 127.0.0.1 | awk '{print $2}'

# en0（通常はWi-Fi）のIPアドレスだけを表示
ifconfig en0 | grep "inet " | awk '{print $2}'

# または
ipconfig getifaddr en0
```

## ⚠️ 注意事項

- ネットワークを変更した場合（別のWi-Fiに接続した場合など）、IPアドレスは変わる可能性があります
- 同じWi-Fiネットワークに接続しているデバイス同士でないと、このIPアドレスにアクセスできません
- ファイアウォールの設定によっては、他のデバイスからアクセスできない場合があります
