# npm install の実行手順

権限エラーが発生している場合、以下の手順を試してください。

## 方法1: ターミナルで直接実行（推奨）

1. **ターミナルを完全に再起動**してください（権限設定を変更した後は必須）

2. **プロジェクトディレクトリに移動**:
   ```bash
   cd "/Users/k.muto/webapp_01/テストデータベース"
   ```

3. **npm installを実行**:
   ```bash
   npm install
   ```

## 方法2: npmのグローバルパスを変更

グローバルなnpmパッケージへのアクセスを回避するため：

```bash
# npmのグローバルパッケージのインストール先を変更
mkdir -p ~/.npm-global
npm config set prefix '~/.npm-global'

# パスを追加
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.zshrc
source ~/.zshrc

# 再度インストール
cd "/Users/k.muto/webapp_01/テストデータベース"
npm install
```

## 方法3: yarnを使用（代替案）

npmの代わりにyarnを使用：

```bash
# yarnをインストール（まだの場合）
brew install yarn

# パッケージをインストール
cd "/Users/k.muto/webapp_01/テストデータベース"
yarn install
```

## 方法4: グローバルnpmの権限を修正

グローバルなnpmパッケージの権限を修正：

```bash
# グローバルnpmディレクトリの所有者を確認
ls -la /usr/local/lib/node_modules/npm/

# 必要に応じて所有者を変更（管理者権限が必要）
sudo chown -R $(whoami) /usr/local/lib/node_modules/npm/
```

## 確認

インストールが成功したら、以下で確認：

```bash
ls node_modules/
```

`node_modules`フォルダが作成され、`pcsclite`と`sqlite3`がインストールされていれば成功です。
