## 🚀 起動手順

1. **ターミナルでプロジェクトディレクトリに移動**
   ```bash
   cd /Users/k.muto/webapp_01
   ```

2. **開発サーバーを起動**
   ```bash
   npm run dev:all
   ```
   または
   ```bash
   ./start-local.sh
   ```

3. **スマホのブラウザでアクセス**
   ```
   http://192.168.43.41:3000
   ```

## ⚠️ 注意事項

- Macとスマホが**同じWi-Fiネットワーク**に接続されている必要があります
- MacのファイアウォールでNode.jsが接続を許可されている必要があります
- ポート3000（フロントエンド）と3001（バックエンド）が他のアプリで使用されていないことを確認してください

## 🔧 接続できない場合の確認事項

1. **Macのファイアウォール設定を確認**
   - システム環境設定 → セキュリティとプライバシー → ファイアウォール
   - Node.jsが接続を許可されているか確認

2. **ポートが使用されているか確認**
   ```bash
   lsof -i :3000
   lsof -i :3001
   ```

3. **Wi-Fiネットワークを確認**
   - Macとスマホが同じWi-Fiに接続されているか確認

4. **IPアドレスが変更されていないか確認**
   ```bash
   ipconfig getifaddr en0
   ```
